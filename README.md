### Summary ###
This is a Python script that generate the Multiplication Table from 1 to 9 in Traditional Chinese.
The output simulates the old school presentation on the back of an exercise book.

這是用作生成「九因歌」的Python執行腳本。
結果模擬傳統練習簿背後的排版。
